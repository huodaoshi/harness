## 文档用途

本文档将产品核心讨论中的用户定义、分层与标签，整理为可指导产品、运营与访谈设计的**核心用户信息**。与《项目建议书》中 MVP 验证用的 Primary Persona（小雨）并存：小雨用于 8 周内测楔子；本文档描述产品长期服务的**核心用户群**与分层策略。

**冲突处理：** 范围、记忆策略、商业模式以最新议会/用户拍板为准；用户画像以产品核心讨论为母本，MVP 场景以建议书 C1–C3 为界。

---

## 一句话定义

**隐忍型高敏感人群**——习惯懂事、报喜不报忧、情绪羞耻感强、不敢麻烦别人、习惯自我消化；渴望被无条件接纳与温柔倾听；拒绝说教与套路；需要安全私密的情绪出口；最终希望获得内生力量、实现自我和解。

**产品承诺（对用户）：** 绝对安全、温柔、不被评判的情绪自留地——安放情绪、认知自己、活出自己。

## 与 MVP 验证用户的关系

| 维度 | 核心用户（本文档） | MVP Persona「小雨」 |

| 场景 | 全疗愈闭环 + 分层认知内容 | **情绪洪峰倾诉** + 关系档案 + 安全护栏 |

## 第一梯队：付费与认知提升主力

### 1. 高压职场女性 / 单亲 / 离异女性

| 字段 | 内容 |
| ---- | ---- |
| **年龄** | 28–45 岁 |
| **典型身份** | 一线城市或新一线白领；单亲/离异独自承担家庭与职场双重压力 |
| **文化与觉察** | 有一定文化水平，自我觉察强，能识别内耗但难以停下 |
| **核心痛点** | 职场内卷、情感创伤、长期内耗；关系议题（原生家庭、伴侣、亲子）与职业压力交织 |
| **情绪模式** | 对外坚强、对内崩溃；习惯自我消化，极少向亲友示弱 |
| **对产品的期待** | 无评判的倾听；温和提问而非说教；逐步看见需求与内在资源 |
| **内容与疗愈** | 适配 NLP 五阶段疗愈；可接受人性、历史类轻认知（L2–L4） |
| **付费特征** | **付费意愿强**，是订阅与增值包的核心来源 |
| **信任敏感点** | 极度反感鸡汤、PUA 式「为你好」、制造依赖的话术 |
| **MVP 关联** | 与 Persona「小雨」高度重叠；关系档案 + 洪峰倾诉优先服务 |

**典型一句话：** 「我知道问题在哪，就是停不下来，也不想再跟人解释一遍。」

---

### 2. 远嫁 / 全职妈妈（受过良好教育）

| 字段 | 内容 |
| ---- | ---- |
| **年龄** | 35–45 岁 |
| **典型身份** | 远嫁或全职育儿；配偶长期缺席情感支持；社交圈窄 |
| **核心痛点** | 家庭琐事繁重、婚姻陪伴缺失、自我价值感低；「被看见、被心疼」的需求长期未满足 |
| **情绪模式** | 愧疚与委屈并存；不敢抱怨，怕被视为「不知足」 |
| **对产品的期待** | 温柔承接委屈；正向意图挖掘；资源唤醒（看见自己曾做到的事） |
| **内容与疗愈** | NLP 正向意图挖掘与资源唤醒；人性轻解读（边界、不配得感） |
| **传播特征** | **口碑传播主力**——信任建立后愿向同圈层密友推荐（非公开社交） |
| **付费特征** | 中等偏上；更愿为「被温柔对待的体验」与写歌/记忆留存付费 |
| **产品注意** | 避免「妈妈应该……」类说教；家庭沟通助手仅提供中立话术，不教经营婚姻 |

**典型一句话：** 「不是日子过不下去，是没有人问过我累不累。」

---

### 3. 高敏感隐忍男性（高学历、职场中坚）

| 字段 | 内容 |
| ---- | ---- |
| **年龄** | 30–55 岁 |
| **典型身份** | 高学历；企业中层或专业岗；社会规训下「不允许脆弱」 |
| **核心痛点** | 工作压力大、情绪羞耻感极强；家庭责任与自我压抑并存 |
| **情绪模式** | 极少倾诉；讨厌鸡汤与说教；偏好逻辑清晰、克制的对话 |
| **对产品的期待** | **提问式梳理**（NLP），而非情感绑架；隐私与界面「干净」 |
| **内容与疗愈** | NLP 提问式梳理；历史/人性复盘（困境与自洽，非励志） |
| **付费特征** | **决策理性**；愿为隐私、无打扰、高品质体验付费，反感套路营销 |
| **产品注意** | 无推送、无红点；本地存储与一键删除是信任前提；忌「暖男 AI」人设 |

**典型一句话：** 「不是没情绪，是说了也没人信，还不如不说。」

### 画像 A：小雨 — 高压职场女性（MVP 主 Persona）

> 与《项目建议书》Primary Persona 对齐，此处补全生活细节。

#### 基本信息

| 项目 | 内容 |
| ---- | ---- |
| **姓名** | 小雨（化名） |
| **年龄** | 32 岁 |
| **城市** | 上海（老家：江苏三线县城） |
| **学历** | 本科，新闻传播 |
| **职业** | 互联网公司品牌运营，司龄 5 年，带 2 人小团队 |
| **婚姻** | 已婚 3 年，暂无孩子（备孕压力来自婆婆） |
| **居住** | 与丈夫租住两居室，周末常回公婆家吃饭 |
| **月收入** | 税前约 2.2 万（家庭主要收入之一） |

#### 家庭与关系脉络

- **母亲：** 爱操心、习惯否定式关心（「你怎么又……」）；小雨报喜不报忧，每周通话 1 次，聊完常更堵。  
- **父亲：** 沉默型，重大决定站母亲一边。  
- **丈夫：** 务实、不擅长情绪话题；吵架后常冷处理，小雨越追越累。  
- **婆婆：** 催生、边界感弱；小雨表面客气，私下委屈积压。  
- **闺蜜：** 2 位大学好友，一个在外地一个刚当妈，**不好意思总占线倾诉**。

#### 日常一天（工作日）

| 时段 | 典型状态 |
| ---- | -------- |
| 7:30 | 刷手机醒神，看工作群，压抑起床气 |
| 9:00–19:00 | 开会、改稿、对上汇报；午餐常边吃边回消息 |
| 20:00 | 回家做饭或外卖；丈夫刷短视频，她收拾厨房 |
| 22:30 | 躺床仍清醒，**翻豆包或小红书**，搜「原生家庭」「边界感」 |
| 23:30 | 自责熬夜，承诺明天早起运动（很少做到） |

#### 情绪与行为模式

- **九型倾向：** 2 号（助人）+ 6 号（焦虑）——同事眼里靠谱，自己知道在硬撑。  
- **崩溃触发：** 被领导当众否定；丈夫说「你想多了」；母亲电话里的指责。  
- **应对方式：** 洗澡哭一场、写备忘录、跟 AI 聊 20–40 分钟；**极少预约咨询**（羞耻 + 觉得「还没严重到那份上」）。  
- **数字习惯：** 豆包高频；测测做过几次测试觉得「不准但好玩」；小红书收藏心理笔记，**很少看完**。

#### 核心痛点（产品语言）

1. 豆包「说得对」，但记不住她和丈夫**第三次因为婆婆吵架**的前因后果。  
2. 想要**倾诉**，不是被教育「你要沟通」「你要感恩」。  
3. 怕聊天记录被看见（丈夫偶尔用同一台 iPad）。  
4. 深夜需要有人**接住当下这一团乱**，而不是给 10 条建议。

#### 与产品的关系（假设旅程）

| 阶段 | 行为 |
| ---- | ---- |
| 发现 | 闺蜜私聊推荐：「这个不会催你买课」 |
| 首次 | 点「我现在很难受」，倾诉婆婆+催生，**不要求填档案**也能聊下去 |
| 留存 | 愿意补「关系档案」：我 / 丈夫 / 婆婆 / 议题「边界与催生」 |
| 付费 | 愿为「写一首歌留住那晚的心情」付小额；反感弹窗和年卡 |
| 警惕 | 若 AI 说「我比任何人都懂你」会立刻卸载 |

#### 典型原话

- 「我不是不知道该怎么办，我是**当下气到发抖**，需要有人先听我说完。」  
- 「跟我妈讲电话 20 分钟，比加一天班还累。」  
- 「豆包像心理咨询师公开课，**我想找的是闺蜜那种房间**。」

#### 访谈深挖清单（小雨）

- [ ] 最近一次洪峰：前 10 分钟身体感受、最先冒出来的话、后来缺什么  
- [ ] 若 AI 记住「婆婆催生」，她希望记住多少、多久、能否一键删  
- [ ] 什么话术会让她觉得被说教？举 3 句原话  
- [ ] 愿为哪些功能付多少钱（写歌 / 音色 / 月度回顾）

### 画像 B：美琳 — 远嫁全职妈妈（口碑传播主力）

#### 基本信息

| 项目 | 内容 |
| ---- | ---- |
| **姓名** | 美琳（化名） |
| **年龄** | 38 岁 |
| **城市** | 现居成都（娘家：山东潍坊） |
| **学历** | 硕士，教育学（婚前在老家中学任教 6 年） |
| **职业** | 全职带娃 4 年（老大 7 岁、老二 3 岁） |
| **婚姻** | 丈夫 40 岁，互联网公司技术管理，**出差每月 8–12 天** |
| **居住** | 婚后随丈夫落户成都，与公婆同城不同住（周末常聚餐） |
| **家庭收入** | 丈夫主挣，她管账；对自己「不挣钱」敏感 |

#### 家庭与关系脉络

- **娘家：** 父母年迈，视频每周 1 次；母亲常说「当初不同意你嫁那么远」——美琳听后愧疚又烦躁。  
- **丈夫：** 负责养家，自认「已经很不错了」；情绪需求回应少，常用「你想太多」「我很累」结束对话。  
- **公婆：** 嘴上疼孙子，实则按老观念插手喂养与管教；美琳**不想撕破脸**，憋在心里。  
- **社交：** 小区妈妈群、学校家委会；表面热络，**真心话只对 1 位老乡闺蜜**说。  
- **自我认同：** 「以前是老师，现在只是某某妈妈」——自我价值感低谷期长。

#### 日常一天

| 时段 | 典型状态 |
| ---- | -------- |
| 6:30 | 做早饭、叫孩子起床，自己常顾不上吃 |
| 8:30 | 送老大上学，带老二买菜或公园 |
| 10:00–15:00 | 家务、辅导作业准备、回复群消息；**偶发崩溃**在卫生间锁门哭 |
| 17:00 | 接孩子、做饭；丈夫若出差则一人扛全流程 |
| 21:00 | 哄睡后刷 30 分钟手机，看育儿/心理短视频，**越刷越焦虑** |
| 22:30 | 躺床想娘家、想「要是还在老家当老师会怎样」 |

#### 情绪与行为模式

- **九型倾向：** 9 号（和平）+ 2 号（助人）——习惯压抑冲突，爆发前已内伤很久。  
- **崩溃触发：** 丈夫出差时老二发烧；公婆当面说「你不会带」；母亲视频里叹气。  
- **应对方式：** 写日记（断断续续）；对老公发过长微信小作文**已读不回**；很少用 AI，觉得「那些东西不懂妈妈」。

#### 核心痛点（产品语言）

1. 不是缺钱，是**长期情感劳动无人看见**。  
2. 需要有人说「你已经做得很好」，而不是「你要体谅老公/婆婆」。  
3. 对「妈妈应该怎样」极其敏感，**拒绝任何育儿 KPI 式话术**。  
4. 希望偶尔**重温自己**：当老师时的成就感、年轻时的喜好（看书、画画，已搁置）。

#### 与产品的关系（假设旅程）

| 阶段 | 行为 |
| ---- | ---- |
| 发现 | 老乡闺蜜私发链接，强调「没有社区、不会晒」 |
| 首次 | 晚上哄睡后，倾诉「今天又被婆婆说不会带娃」 |
| 关键体验 | AI 做**正向意图挖掘**：「你这么坚持接送，你在保护什么？」——她第一次哭完觉得轻松 |
| 留存 | 每周 2–3 次，单次 15–25 分钟；喜欢**写歌沉淀**某次情绪 |
| 传播 | 只推荐给 1–2 位同样远嫁/全职的朋友，**不发朋友圈** |
| 付费 | 愿付「妈妈自己的声音」记忆包；不愿买「婚姻修复课」 |

#### 典型原话

- 「日子能过，就是**没人问过我累不累**。」  
- 「我不是恨婆婆，我是恨自己**不敢说**。」  
- 「我不想当完美妈妈，我想当**还被当成一个人的美琳**。」

#### 访谈深挖清单（美琳）

- [ ] 「被看见」对她具体指什么：一句话？一种语气？还是不被建议？  
- [ ] 丈夫/公婆/娘家三条线，她最愿先倾诉哪条？为什么  
- [ ] 对「资源唤醒」（回忆自己曾做到的事）是否觉得假？要什么例子才信  
- [ ] 带娃场景下使用：时长、场所、怕不被谁发现

### 画像 C：志远 — 高敏感隐忍男性（理性付费型）

#### 基本信息

| 项目 | 内容 |
| ---- | ---- |
| **姓名** | 志远（化名） |
| **年龄** | 41 岁 |
| **城市** | 深圳 |
| **学历** | 硕士，计算机 |
| **职业** | 制造业企业 IT 总监，团队 15 人 |
| **婚姻** | 已婚 12 年，一子 9 岁（小学） |
| **居住** | 自有住房，与父母同城不同住 |
| **收入** | 家庭年入约 80 万，对订阅价格敏感但**愿为品质付溢价** |

#### 家庭与关系脉络

- **父亲：** 严厉、很少表扬；志远从小学会「别添麻烦」。  
- **母亲：** 情绪化，曾在他青春期说过重话，至今仍是**隐秘心结**（不对妻子细说）。  
- **妻子：** 务实能干，觉得他「冷漠」；争吵点多在育儿分工与陪伴时间。  
- **儿子：** 成绩压力；志远想温柔但开口就变成说教，事后懊悔。  
- **朋友：** 几乎没有「可以哭」的哥们；偶尔喝酒聊球，不聊家里。

#### 日常一天

| 时段 | 典型状态 |
| ---- | -------- |
| 7:00 | 看行业新闻、回邮件 |
| 9:00–20:00 | 开会、救火、对上；午休常独自吃饭 |
| 20:30 | 陪作业，忍住不发火 |
| 22:00 | 书房待 1 小时，刷技术论坛或新闻，**不刷情感类内容** |
| 23:30 | 失眠时坐客厅，**不会叫醒妻子** |

#### 情绪与行为模式

- **九型倾向：** 6 号（忠诚）+ 1 号（完美）——外表稳定，内在反刍多。  
- **崩溃触发：** 项目重大失误被老板点名；儿子说「你从来不理解我」；父亲住院自己却无法请假。  
- **应对方式：** 跑步、独处；曾试过冥想 App，**坚持不到两周**；未用过心理 App，觉得「那是女人用的」。

#### 核心痛点（产品语言）

1. **情绪羞耻**：认为「男人不该软弱」，但身体会用失眠、胃痛报警。  
2. 讨厌鸡汤和「你要积极」；接受**结构化提问**（今天具体发生了什么？你想要什么？）。  
3. **隐私高于一切**：绝不允许推送、红点、微信分享诱导。  
4. 家庭议题集中在：父亲关系、儿子沟通、妻子觉得他冷漠的恶性循环。

#### 与产品的关系（假设旅程）

| 阶段 | 行为 |
| ---- | ---- |
| 发现 | 深夜搜索「压力大 不想跟人说」看到垂直产品评测（非广告软文） |
| 首次 | 试探性短句：「最近失眠」，观察 AI 是否**立刻给建议**——若说教则退出 |
| 信任建立 | 第 3 次会话，才提到父亲；喜欢**提问式梳理**而非共情堆砌 |
| 留存 | 每周 1–2 次，单次 10–20 分钟；偏好文字，**暂不用语音** |
| 付费 | 一次性买断「记忆留存」或写歌导出；**拒绝自动续费** |
| 警惕 | 粉色 UI、称呼「亲」、任何像虚拟女友的互动 |

#### 典型原话

- 「不是没情绪，是**说了也没人信**，还不如不说。」  
- 「我不要你安慰我，我要你**帮我把事情理清楚**。」  
- 「要是这东西会推消息、搞签到，我绝对不会用第二次。」

#### 访谈深挖清单（志远）

- [ ] 男性身份下，哪些词会让他觉得「被冒犯」或「被小看」  
- [ ] 可接受的共情句式 vs 不可接受的鸡汤句式（各 5 句）  
- [ ] 父亲/儿子/妻子三条线，哪条最先愿意对 AI 开口  
- [ ] 本地存储、无账号、假名注册等隐私方案的可信度排序

## 第二梯队：浅层适配、强约束

### 4. 空巢 / 独居长辈（多为母亲）

| 字段 | 内容 |
| ---- | ---- |
| **年龄** | 60–70 岁 |
| **典型身份** | 子女远嫁或联系少；独居或老两口的日常 |
| **核心痛点** | 孤独、愧疚、认知固化；对复杂引导耐心有限 |
| **产品能力边界** | **仅**纯倾听与情绪安抚；极浅和解类小故事、温柔短句 |
| **明确不做** | 深度 NLP 引导、复杂认知内容、长会话引导 |
| **延伸触点** | 怀旧回忆沉淀（本地导出）；晚辈视角温柔短句（可选） |

**典型一句话：** 「就想有人说说话，别讲大道理。」

### 5. 高压青少年 / 大学生

| 字段 | 内容 |
| ---- | ---- |
| **年龄** | 16–24 岁 |
| **典型身份** | 学业高压；原生家庭矛盾；自我否定明显 |
| **核心痛点** | 情绪急救需求高；对「被理解」敏感，对说教更敏感 |
| **产品能力边界** | 纯倾听情绪急救；轻认知短句、自我接纳类内容 |
| **强制约束** | **严格时长管控、强反成瘾**；禁止深度引导；内容双审核 |
| **明确不做** | 虚拟恋爱向、暧昧陪伴、成瘾式高频互动 |

**典型一句话：** 「只想今晚能睡着，不想听谁教育我。」

## 绝对排除用户

以下用户群会稀释产品调性、抬高内容与安全风险，**不作为目标用户**，运营与推荐上不主动获取：

| 类型 | 特征 | 排除原因 |
| ---- | ---- | -------- |
| 泛娱乐闲聊 / 暧昧猎奇 | 把 AI 当虚拟恋人或撩骚对象 | 成瘾、审核与伦理风险 |
| 快餐式情绪用户 | 追求强刺激、即时爽感、刷屏式发泄 | 与「温和、克制、疗愈」相悖 |
| 精神依赖 / PUA 受众 | 寻求洗脑式依赖、重度鸡汤、心灵控制 | 二次伤害、商业化伦理问题 |

## 需求与动机矩阵

| 需求层次 | 用户原话（归纳） | 产品回应 |
| -------- | ---------------- | -------- |
| **安全** | 「不能说出去」「别被评判」 | 本地存储、不上传训练、一键删除、危机 SOP |
| **被接住** | 「接不住我的具体情况」 | 关系档案、记忆摘要（Level 1–2 起）、倾听话术 |
| **被看见** | 「没人知道我有多累」 | 正向意图挖掘、资源唤醒、写歌沉淀 |
| **认知自己** | 「总重复同样的痛」 | 人性/历史轻解读、自我故事梳理（L3+） |
| **回归生活** | 「聊完还得过日子」 | 微行动、沟通话术工具；引导看见现实资源 |
| **希望感** | 「想知道有人走过」 | 微光故事集（匿名、无互动、L4/L5） |

**共性拒绝：** 说教、收割、二次伤害、成瘾设计、虚拟恋人替代现实。

## 核心使用场景

### 场景 A：情绪洪峰（当晚，全梯队）

- **触发：** 吵架、被催婚、翻旧账、职场崩溃、孤独爆发  
- **时长：** 15–30 分钟  
- **目标：** 倾诉 → 锚定 → 平复（L1→L2）；MVP 聚焦此楔子  
- **成功信号：** 「说出来轻了一点」「没有更糟」

### 场景 B：关系复盘（周末，第一梯队，v1.1+）

- **触发：** 反复踩雷的模式、边界不清  
- **目标：** 理清模式与边界（**非诊断**）  
- **与 MVP：** 轻复盘默认关闭，记忆会议后再定摘要深度  

### 场景 C：日常微支持（可选）

- **触发：** 防积压成洪峰  
- **约束：** 无打卡、无任务、无推送诱导  

### 场景 D：认知滋养（第一梯队 L2–L4）

- **内容：** 800 字内人性解读、历史人物复盘、自我故事短文  
- **原则：** 不鸡汤、不说教、不强行改造价值观

## 典型用户旅程（第一梯队）

```text
发现（密友推荐 / 主动搜索「安全倾诉」）
  → 首次洪峰（纯倾听，建立信任）
  → 重复来访（锚定 + 意图挖掘）
  → L3 觉察（轻认知、写歌）
  → 可选增值（音色、写歌定制、记忆留存）
  → L4/L5（微光阅读或匿名分享，闭环）
  → 回归现实（微行动，主动降低使用频率）
```

**反成瘾节点：** 单日 8 次或连续 7 天高频 → 自动降级纯倾听；长期记忆 30 天后淡化细节。

## 访谈与内测筛选建议

**纳入（对齐 MVP）：**

- 25–40 岁，原生家庭/伴侣/亲子议题明确  
- 用过豆包或泛心理 App，仍感「接不住」  
- 未在接受重度临床治疗，但自我觉察中等以上  

**优先深挖（第一梯队）：** 对照上文具象画像招募，建议配额：

| 画像 | 招募人数 | 关键匹配条件 |
| ---- | -------- | ------------ |
| **小雨 A** | 4–6 人 | 25–35 岁、一线/新一线、职场+亲密关系议题、用过豆包 |
| **美琳 B** | 2–3 人 | 35–42 岁、全职或半职妈妈、远嫁或配偶长期缺席、自我价值感议题 |
| **志远 C** | 2–3 人 | 35–50 岁、男性、高学历职场、极少向人倾诉、重视隐私 |

每人访谈 60–90 分钟，含 1 次**纸面/原型洪峰路径**走查（见各画像「访谈深挖清单」）。

**暂缓（第二梯队）：**

- 16–24 岁、60+ 岁——待 MVP 门禁与青少年策略定稿后再开专项  

**筛除：**

- 明确寻找虚拟恋爱、算命、情感挽回者  

**建议筛选题（示例）：**

1. 最近一次情绪崩溃时，你最先找谁？为什么没找或找完仍难受？  
2. 你是否介意「被 AI 记住」关系细节？边界在哪里？  
3. 你更害怕：被说教 / 被泄露 / 被诱导多聊？
