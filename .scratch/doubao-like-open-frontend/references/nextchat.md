# 参考：NextChat（原 ChatGPT-Next-Web）

**类型：** 直接竞品
**链接：** https://github.com/ChatGPTNextWeb/NextChat
**来源：** Agent 搜索
**置信度：** 高
**搜索日期：** 2026-05-19

## 他们解决什么

轻量、跨端的 ChatGPT 式对话前端，一键部署，多模型配置。

## 目标用户

个人用户、需要快速私有部署 Chat 界面的开发者。

## 核心功能（与本次想法相关的）

- 流式 SSE 对话、多会话、Prompt 模板
- 支持 OpenAI 兼容 API 及多家模型（含国内常用）
- Web + Tauri 桌面（Win/macOS/Linux）+ 移动端
- PWA、本地设置同步（视部署方式）

## 定价 / 商业模式

MIT 开源；无强制商业组件。

## 优点

- Star 极高、中文文档与社区成熟
- 部署简单（Vercel/Docker），适合「只要聊天壳」

## 缺点 / 局限

- UI 偏 ChatGPT 经典侧栏，与豆包「信息流/多入口」差异大
- Agent/RAG/知识库需自行扩展或接后端
