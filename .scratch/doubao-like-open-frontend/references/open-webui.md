# 参考：Open WebUI

**类型：** 直接竞品
**链接：** https://github.com/open-webui/open-webui
**来源：** Agent 搜索
**置信度：** 高
**搜索日期：** 2026-05-19

## 他们解决什么

自托管「AI 平台」：聊天 + 模型管理 + RAG + 工具/管道，偏一体化运维。

## 目标用户

要在内网/私有云跑完整 AI 工作台的企业或进阶个人。

## 核心功能（与本次想法相关的）

- Ollama / OpenAI 兼容 API 对话，会话内换模型
- 知识库、RAG、Agent、代码执行、网页搜索（视版本）
- 用户/RBAC、SSO（企业向）
- 语音、图像生成等扩展

## 定价 / 商业模式

开源（许可证以仓库为准）；有商业/企业版路线（未找到公开定价细节）。

## 优点

- 功能最全之一，接近「豆包 + 后台能力」一体化
- Docker 一键、中文文档站（openwebui.cn）完善

## 缺点 / 局限

- 前后端一体，**纯前端**拆出成本高
- 体量大，只要 UI 壳可能过重

## 与我们的关系

间接正面竞争 — 要「平台」选它；要「可嵌入前端组件」需评估拆分。
