## 他们解决什么

基于 Nuxt 3 的自托管聊天 UI，主打 Ollama，并扩展多家云 API。

## 目标用户

本地模型 +  occasional 云 API；要知识库与 Agent 的进阶用户。

## 核心功能（与本次想法相关的）

- 流式对话、多模型（OpenAI/Anthropic/Gemini/Groq/Ollama 等）
- 知识库、MCP、Agent / Deep Agents（版本功能以仓库为准）
- 语音对话、Docker 部署

## 定价 / 商业模式

开源；未找到公开定价。

## 优点

- 本地优先，隐私场景友好
- 功能面介于 NextChat 与 Lobe 之间

## 缺点 / 局限

- 品牌与豆包 C 端审美有差距
- 国内文档相对少

## 与我们的关系

支持借鉴 — 偏「本地助手」路线；豆包式云助手需自己接 API。
