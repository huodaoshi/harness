# 参考：Lobe Chat（LobeHub）

**类型：** 直接竞品
**链接：** https://github.com/lobehub/lobe-chat
**来源：** Agent 搜索
**置信度：** 高
**搜索日期：** 2026-05-19

## 他们解决什么

开箱即用的现代 AI 对话 Web/桌面壳，多模型、插件、知识库、语音与多模态。

## 目标用户

要漂亮 UI + 多厂商 API + 可自托管的个人/小团队。

## 核心功能（与本次想法相关的）

- 流式对话、会话管理、多模型切换
- 文件/知识库、TTS/STT、文生图（视配置）
- MCP、Function Calling、插件市场
- PWA / Electron 桌面端
- 中文社区活跃，文档与部署教程多

## 定价 / 商业模式

MIT 开源；官方有云服务（未核实具体定价）。

## 优点

- 界面接近国内「助手 App」审美，豆包用户上手快
- 生态大、迭代快、TypeScript/Next.js 栈主流

## 缺点 / 局限

- 功能多 -> 定制/裁剪成本高
- 默认偏「接各家 API」，深度绑豆包需自行接 Volcengine/兼容层

## 与我们的关系

正面竞争 / 首选 fork 或参考 UI 之一 — 最接近「现代助手壳」全能型。
